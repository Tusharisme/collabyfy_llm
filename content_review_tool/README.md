# AI Content Review Tool

This tool acts as an automated "Editor-in-Chief," using **Gemini 2.5 Flash (Multimodal)** to analyze images and captions together. It provides instant feedback on tone, quality, and ad compliance.

## Features
- **Multimodal Understanding**: Checks if the caption matches the image context.
- **Tone Analysis**: Rates engagement and "viral potential."
- **Compliance Check**: Detects undisclosed advertisements.
- **Image Quality**: Scores lighting, focus, and composition.

## Setup
1.  **Prerequisites**: Python 3.9+
2.  **Installation**:
    ```bash
    pip install -r requirements.txt
    ```
3.  **Environment Variables**:
    Create a `.env` file in this directory:
    ```ini
    API_KEY=your_gemini_api_key_here
    ```

## Usage
Run the CLI to analyze a specific image and caption:
```bash
python main.py --image "path/to/photo.jpg" --caption "Your caption here"
```

## Integration Guide (Collabyfy App)

To filter high-quality content in your main app:
1.  **Worker Service**: Wrap this tool in a background worker (e.g., Celery/BullMQ).
2.  **Trigger**: When a creator uploads a draft post.
3.  **Feedback Loop**:
    - **Score > 80**: Auto-approve for brand review.
    - **Score < 50**: Return to creator with the specific tips generated by the tool.
